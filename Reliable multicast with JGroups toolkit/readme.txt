The program contains five classes.

VSynchrony is the main class where running code is implemented.

Stock class records the information for each stock, including name, symbol, price, orderlists and methods like adding incoming orders into the corresponding buylist or selllist, and sort the two lists.

Order class contains the type, sender, stock name, stock symbol, price and share information.

Account class provides information of every client, and methods for conducting the client balance.

StateOb class is the class used for state recording and state transfer, which includes stock information and account information, as well as current order number.
